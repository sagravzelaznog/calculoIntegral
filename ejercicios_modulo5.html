Variables Separables

1
Separación Básica
Fácil

Resolver: $\frac{dy}{dx} = \frac{x^2}{y}$

Pista

Separa las variables y luego integra ambos lados. Recuerda la constante de integración.

1 Separar variables:

$$\frac{dy}{dx} = \frac{x^2}{y}$$

$$y \, dy = x^2 \, dx$$

2 Integrar ambos lados:

$$\int y \, dy = \int x^2 \, dx$$

$$\frac{y^2}{2} = \frac{x^3}{3} + C$$

3 Despejar y:

$$y^2 = \frac{2x^3}{3} + 2C$$

$$\boxed{y = \pm\sqrt{\frac{2x^3}{3} + C}}$$

2
Con Condición Inicial
Medio

Resolver: $\frac{dy}{dx} = \frac{y^2}{x}$ con $y(1) = 2$

Pista

Separa las variables, integra, y luego usa la condición inicial para encontrar la constante.

1 Separar variables:

$$\frac{dy}{dx} = \frac{y^2}{x}$$

$$\frac{dy}{y^2} = \frac{dx}{x}$$

2 Integrar:

$$\int \frac{dy}{y^2} = \int \frac{dx}{x}$$

$$-\frac{1}{y} = \ln|x| + C$$

3 Aplicar condición inicial:

$y(1) = 2 \Rightarrow -\frac{1}{2} = \ln|1| + C = 0 + C$

$$C = -\frac{1}{2}$$

4 Solución particular:

$$-\frac{1}{y} = \ln|x| - \frac{1}{2}$$

$$\frac{1}{y} = \frac{1}{2} - \ln|x|$$

$$\boxed{y = \frac{1}{\frac{1}{2} - \ln|x|}}$$

Ecuaciones Homogéneas

3
Homogénea Básica
Medio

Resolver: $\frac{dy}{dx} = \frac{x^2 + y^2}{2xy}$

Pista

Verifica que es homogénea y usa la sustitución $y = vx$ donde $v$ es una función de $x$.

1 Verificar homogeneidad:

$$\frac{dy}{dx} = \frac{x^2 + y^2}{2xy} = \frac{x^2(1 + \frac{y^2}{x^2})}{2xy} = \frac{1 + \left(\frac{y}{x}\right)^2}{2\left(\frac{y}{x}\right)}$$

Es homogénea ya que depende solo de $\frac{y}{x}$.

2 Sustitución:

$y = vx \Rightarrow \frac{dy}{dx} = v + x\frac{dv}{dx}$

$$v + x\frac{dv}{dx} = \frac{1 + v^2}{2v}$$

3 Separar variables:

$$x\frac{dv}{dx} = \frac{1 + v^2}{2v} - v = \frac{1 + v^2 - 2v^2}{2v} = \frac{1 - v^2}{2v}$$

$$\frac{2v}{1-v^2} dv = \frac{dx}{x}$$

4 Integrar:

$$\int \frac{2v}{1-v^2} dv = \int \frac{dx}{x}$$

$$-\ln|1-v^2| = \ln|x| + C$$

$$\ln|x(1-v^2)| = C$$

$$x(1-v^2) = A$$

5 Solución final:

$$x\left(1 - \frac{y^2}{x^2}\right) = A$$

$$\boxed{x^2 - y^2 = Ax}$$

Ecuaciones Exactas

4
Exacta Básica
Medio

Resolver: $(2xy + 1)dx + (x^2 - 2y)dy = 0$

Pista

Primero verifica que la ecuación sea exacta usando la condición $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$.

1 Verificar exactitud:

$M(x,y) = 2xy + 1$ y $N(x,y) = x^2 - 2y$

$$\frac{\partial M}{\partial y} = 2x$$

$$\frac{\partial N}{\partial x} = 2x$$

Como $\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$, la ecuación es exacta. ✓

2 Encontrar F(x,y):

$$\frac{\partial F}{\partial x} = M = 2xy + 1$$

$$F = \int (2xy + 1)dx = x^2y + x + g(y)$$

3 Determinar g(y):

$$\frac{\partial F}{\partial y} = x^2 + g'(y) = N = x^2 - 2y$$

$$g'(y) = -2y \Rightarrow g(y) = -y^2 + C$$

$$F(x,y) = x^2y + x - y^2 + C$$

4 Solución general:

$$\boxed{x^2y + x - y^2 = C}$$

Ecuaciones Lineales de Primer Orden

5
Factor Integrante
Medio

Resolver: $\frac{dy}{dx} + 3y = e^{2x}$

Pista

Usa el factor integrante $\mu(x) = e^{\int P(x)dx}$ donde $P(x) = 3$.

1 Identificar P(x) y Q(x):

$P(x) = 3$ y $Q(x) = e^{2x}$

2 Calcular factor integrante:

$$\mu(x) = e^{\int P(x)dx} = e^{\int 3 dx} = e^{3x}$$

3 Aplicar factor integrante:

$$e^{3x}\frac{dy}{dx} + 3e^{3x}y = e^{3x}e^{2x} = e^{5x}$$

$$\frac{d}{dx}(ye^{3x}) = e^{5x}$$

4 Integrar:

$$ye^{3x} = \int e^{5x} dx = \frac{e^{5x}}{5} + C$$

$$\boxed{y = \frac{e^{2x}}{5} + Ce^{-3x}}$$

6
Con Condición Inicial
Difícil

Resolver: $xy' + 2y = x^2$ con $y(1) = 1$

Pista

Primero convierte a la forma estándar dividiendo por $x$, luego usa el factor integrante.

1 Forma estándar:

$$xy' + 2y = x^2$$

$$y' + \frac{2}{x}y = x$$

$P(x) = \frac{2}{x}$ y $Q(x) = x$

2 Factor integrante:

$$\mu(x) = e^{\int \frac{2}{x} dx} = e^{2\ln|x|} = x^2$$

3 Aplicar factor integrante:

$$x^2y' + 2xy = x^3$$

$$\frac{d}{dx}(yx^2) = x^3$$

4 Integrar:

$$yx^2 = \int x^3 dx = \frac{x^4}{4} + C$$

$$y = \frac{x^2}{4} + \frac{C}{x^2}$$

5 Aplicar condición inicial:

$y(1) = 1 \Rightarrow 1 = \frac{1}{4} + C \Rightarrow C = \frac{3}{4}$

$$\boxed{y = \frac{x^2}{4} + \frac{3}{4x^2}}$$

Aplicaciones

7
Crecimiento Poblacional
Medio

Problema: Una población de conejos crece a una tasa proporcional a su tamaño. Si inicialmente hay 50 conejos y después de 1 año hay 100, ¿cuántos habrá después de 3 años?

Pista

Usa el modelo $\frac{dP}{dt} = kP$ donde $P(t)$ es la población en el tiempo $t$.

1 Modelo matemático:

$$\frac{dP}{dt} = kP$$

donde $P(t)$ es la población en el tiempo $t$ (en años).

2 Resolver la ecuación:

$$\frac{dP}{P} = k \, dt$$

$$\ln|P| = kt + C$$

$$P = A e^{kt}$$

3 Aplicar condiciones:

$P(0) = 50 \Rightarrow A = 50$

$P(1) = 100 \Rightarrow 100 = 50 e^{k}$

$$e^{k} = 2 \Rightarrow k = \ln 2$$

4 Respuesta:

$$P(t) = 50 e^{t \ln 2} = 50 \cdot 2^t$$

Para $t = 3$: $P(3) = 50 \cdot 2^3 = 50 \cdot 8 = 400$

Después de 3 años habrá 400 conejos.

8
Ley de Enfriamiento
Difícil

Problema: Una taza de café a 95°C se coloca en una habitación a 20°C. Después de 10 minutos, la temperatura es 70°C. ¿Cuánto tiempo tardará en llegar a 40°C?

Pista

Usa la ley de enfriamiento de Newton: $\frac{dT}{dt} = -k(T - T_a)$ donde $T_a$ es la temperatura ambiente.

1 Modelo matemático:

$$\frac{dT}{dt} = -k(T - 20)$$

donde $T$ es la temperatura del café y $T_a = 20°C$.

2 Resolver:

Sustitución: $u = T - 20 \Rightarrow \frac{du}{dt} = \frac{dT}{dt}$

$$\frac{du}{dt} = -ku$$

$$u = A e^{-kt}$$

$$T = 20 + A e^{-kt}$$

3 Aplicar condiciones:

$T(0) = 95 \Rightarrow 95 = 20 + A \Rightarrow A = 75$

$T(10) = 70 \Rightarrow 70 = 20 + 75e^{-10k}$

$$50 = 75e^{-10k} \Rightarrow e^{-10k} = \frac{2}{3}$$

$$k = -\frac{\ln(2/3)}{10}$$

4 Encontrar tiempo para 40°C:

$$40 = 20 + 75e^{-kt}$$

$$20 = 75e^{-kt} \Rightarrow e^{-kt} = \frac{4}{15}$$

$$-kt = \ln\left(\frac{4}{15}\right)$$

$$t = \frac{\ln(15/4)}{k} = \frac{10 \ln(15/4)}{\ln(3/2)} \approx 37.7 \text{ minutos}$$
