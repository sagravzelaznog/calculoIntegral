Módulo 1: Antiderivadas e Integrales Indefinidas

Objetivo: Comprender antiderivadas como operación inversa de la derivada

Progreso del Módulo

0% completado Tiempo: 0 min

1.1 Concepto de Antiderivada

2 horas Básico

Introducción

En Cálculo Diferencial aprendiste que la derivada de una función representa la razón de cambio instantánea. Ahora vamos a explorar la operación inversa: la antiderivada.

Imagina que tienes la velocidad de un objeto en cada momento. ¿Cómo podrías encontrar su posición? ¡Exacto! Necesitas "deshacer" la derivación.

Definición de Antiderivada

Definición: Una función $F(x)$ es una antiderivada de $f(x)$ en un intervalo $I$ si

$$F'(x) = f(x) \text{ para todo } x \text{ en } I$$

En otras palabras, la derivada de $F(x)$ es $f(x)$.

Ejemplo 1: Antiderivada Básica

Problema: Encuentra una antiderivada de $f(x) = 3x^2$

Paso 1: Recordar las reglas de derivación

Sabemos que $\frac{d}{dx}[x^3] = 3x^2$

Paso 2: Identificar la antiderivada

Si $\frac{d}{dx}[x^3] = 3x^2$, entonces $F(x) = x^3$ es una antiderivada de $f(x) = 3x^2$

Paso 3: Verificar

Verificamos: $\frac{d}{dx}[x^3] = 3x^2$ ✓

Respuesta: Una antiderivada de $f(x) = 3x^2$ es $F(x) = x^3$

La Constante de Integración

¡Importante!

Las antiderivadas no son únicas. Si $F(x)$ es una antiderivada de $f(x)$, entonces $F(x) + C$ también lo es, donde $C$ es cualquier constante.

¿Por qué?

Porque $\frac{d}{dx}[F(x) + C] = F'(x) + 0 = F'(x) = f(x)$

La derivada de cualquier constante es cero, por lo que agregar una constante no cambia la derivada.

Ejemplo 2: Familia de Antiderivadas

Problema: Encuentra todas las antiderivadas de $f(x) = 2x$

Paso 1: Encontrar una antiderivada

Sabemos que $\frac{d}{dx}[x^2] = 2x$, entonces $F(x) = x^2$ es una antiderivada.

Paso 2: Agregar la constante

Todas las antiderivadas tienen la forma: $F(x) = x^2 + C$ donde $C$ es cualquier constante.

Paso 3: Visualizar la familia

La familia de antiderivadas $F(x) = x^2 + C$ representa una familia de parábolas:

  • $C = 0$: $F(x) = x^2$ (pasa por el origen)
  • $C = 2$: $F(x) = x^2 + 2$ (desplazada 2 unidades hacia arriba)
  • $C = -1$: $F(x) = x^2 - 1$ (desplazada 1 unidad hacia abajo)

Respuesta: Todas las antiderivadas de $f(x) = 2x$ son $F(x) = x^2 + C$

Notación de la Integral Indefinida

Para representar todas las antiderivadas de $f(x)$, usamos la notación:

$$\int f(x) \, dx = F(x) + C$$

Donde:

  • $\int$ es el símbolo de integral
  • $f(x)$ es el integrando
  • $dx$ indica la variable de integración
  • $F(x)$ es una antiderivada de $f(x)$
  • $C$ es la constante de integración

Ejemplo 3: Usando la Notación de Integral

Problema: Evalúa $\int 4x^3 \, dx$

Paso 1: Identificar el integrando

El integrando es $f(x) = 4x^3$

Paso 2: Encontrar una antiderivada

Sabemos que $\frac{d}{dx}[x^4] = 4x^3$, entonces $F(x) = x^4$ es una antiderivada.

Paso 3: Escribir la respuesta

$$\int 4x^3 \, dx = x^4 + C$$

Paso 4: Verificar

Verificamos: $\frac{d}{dx}[x^4 + C] = 4x^3 + 0 = 4x^3$ ✓

Respuesta: $\int 4x^3 \, dx = x^4 + C$

Resumen de Conceptos Clave

  • Antiderivada: Una función cuya derivada es la función dada
  • Constante de integración: Las antiderivadas no son únicas; difieren por una constante
  • Notación: $\int f(x) \, dx = F(x) + C$ representa todas las antiderivadas
  • Verificación: Siempre verifica tu respuesta derivando

1.2 Integrales Inmediatas

3 horas Básico

Introducción

Las integrales inmediatas son aquellas que podemos resolver directamente aplicando las reglas básicas de integración. Son el equivalente inverso de las reglas de derivación que ya conoces.

Tabla de Integrales Inmediatas

Función Integral Condición
$x^n$ $\frac{x^{n+1}}{n+1} + C$ $n \neq -1$
$\frac{1}{x}$ $\ln|x| + C$ $x \neq 0$
$e^x$ $e^x + C$ -
$a^x$ $\frac{a^x}{\ln a} + C$ $a > 0, a \neq 1$
$\sin x$ $-\cos x + C$ -
$\cos x$ $\sin x + C$ -
$\sec^2 x$ $\tan x + C$ -
$\csc^2 x$ $-\cot x + C$ -

Regla de la Potencia para Integrales

$$\int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad \text{si } n \neq -1$$

Esta es la regla más importante y la usaremos constantemente.

Ejemplo 4: Regla de la Potencia

Problema: Evalúa $\int x^5 \, dx$

Paso 1: Identificar n

En $x^5$, tenemos $n = 5$

Paso 2: Aplicar la regla

$\int x^5 \, dx = \frac{x^{5+1}}{5+1} + C = \frac{x^6}{6} + C$

Paso 3: Verificar

$\frac{d}{dx}\left[\frac{x^6}{6} + C\right] = \frac{6x^5}{6} = x^5$ ✓

Respuesta: $\int x^5 \, dx = \frac{x^6}{6} + C$

Ejemplo 5: Potencias Fraccionarias

Problema: Evalúa $\int \sqrt{x} \, dx$

Paso 1: Reescribir como potencia

$\sqrt{x} = x^{1/2}$

Paso 2: Aplicar la regla

$\int x^{1/2} \, dx = \frac{x^{1/2+1}}{1/2+1} + C = \frac{x^{3/2}}{3/2} + C$

Paso 3: Simplificar

$\frac{x^{3/2}}{3/2} = \frac{2x^{3/2}}{3}$

Paso 4: Reescribir en forma radical

$x^{3/2} = x \cdot x^{1/2} = x\sqrt{x}$

Respuesta: $\int \sqrt{x} \, dx = \frac{2x\sqrt{x}}{3} + C$

Integrales Trigonométricas Básicas

$$\int \sin x \, dx = -\cos x + C$$ $$\int \cos x \, dx = \sin x + C$$ $$\int \sec^2 x \, dx = \tan x + C$$ $$\int \csc^2 x \, dx = -\cot x + C$$

Ejemplo 6: Integrales Trigonométricas

Problema: Evalúa $\int (3\sin x + 2\cos x) \, dx$

Paso 1: Separar las integrales

$\int (3\sin x + 2\cos x) \, dx = 3\int \sin x \, dx + 2\int \cos x \, dx$

Paso 2: Aplicar las fórmulas

$3(-\cos x) + 2(\sin x) + C = -3\cos x + 2\sin x + C$

Paso 3: Verificar

$\frac{d}{dx}[-3\cos x + 2\sin x + C] = 3\sin x + 2\cos x$ ✓

Respuesta: $\int (3\sin x + 2\cos x) \, dx = -3\cos x + 2\sin x + C$

Propiedades de Linealidad

$$\int [f(x) + g(x)] \, dx = \int f(x) \, dx + \int g(x) \, dx$$ $$\int k \cdot f(x) \, dx = k \int f(x) \, dx$$

Estas propiedades nos permiten separar integrales complejas en integrales más simples.

Ejemplo 7: Aplicando Propiedades de Linealidad

Problema: Evalúa $\int (4x^3 - 2x^2 + 5x - 1) \, dx$

Paso 1: Separar término por término

$\int (4x^3 - 2x^2 + 5x - 1) \, dx = 4\int x^3 \, dx - 2\int x^2 \, dx + 5\int x \, dx - \int 1 \, dx$

Paso 2: Aplicar la regla de la potencia

$4 \cdot \frac{x^4}{4} - 2 \cdot \frac{x^3}{3} + 5 \cdot \frac{x^2}{2} - x + C$

Paso 3: Simplificar

$x^4 - \frac{2x^3}{3} + \frac{5x^2}{2} - x + C$

Respuesta: $\int (4x^3 - 2x^2 + 5x - 1) \, dx = x^4 - \frac{2x^3}{3} + \frac{5x^2}{2} - x + C$

1.3 Técnicas Básicas de Integración

3 horas Intermedio

Introducción

No todas las integrales son inmediatas. A veces necesitamos reescribir la función antes de poder integrarla usando las reglas básicas.

Reescritura de Funciones

Técnicas de Reescritura:

  • Expansión: Multiplicar expresiones
  • División: Dividir numerador entre denominador
  • Factorización: Factorizar expresiones comunes
  • Identidades: Usar identidades algebraicas o trigonométricas

Ejemplo 8: Expansión

Problema: Evalúa $\int (x + 2)^2 \, dx$

Paso 1: Expandir el binomio

$(x + 2)^2 = x^2 + 4x + 4$

Paso 2: Integrar término por término

$\int (x^2 + 4x + 4) \, dx = \int x^2 \, dx + 4\int x \, dx + 4\int 1 \, dx$

Paso 3: Aplicar reglas básicas

$= \frac{x^3}{3} + 4 \cdot \frac{x^2}{2} + 4x + C$

Paso 4: Simplificar

$= \frac{x^3}{3} + 2x^2 + 4x + C$

Respuesta: $\int (x + 2)^2 \, dx = \frac{x^3}{3} + 2x^2 + 4x + C$

Ejemplo 9: División de Polinomios

Problema: Evalúa $\int \frac{x^3 + 2x^2 - 1}{x} \, dx$

Paso 1: Dividir término por término

$\frac{x^3 + 2x^2 - 1}{x} = \frac{x^3}{x} + \frac{2x^2}{x} - \frac{1}{x} = x^2 + 2x - \frac{1}{x}$

Paso 2: Integrar término por término

$\int (x^2 + 2x - \frac{1}{x}) \, dx = \int x^2 \, dx + 2\int x \, dx - \int \frac{1}{x} \, dx$

Paso 3: Aplicar reglas

$= \frac{x^3}{3} + 2 \cdot \frac{x^2}{2} - \ln|x| + C$

Paso 4: Simplificar

$= \frac{x^3}{3} + x^2 - \ln|x| + C$

Respuesta: $\int \frac{x^3 + 2x^2 - 1}{x} \, dx = \frac{x^3}{3} + x^2 - \ln|x| + C$

Ejemplo 10: Usando Identidades Trigonométricas

Problema: Evalúa $\int \sin^2 x \, dx$

Paso 1: Usar identidad trigonométrica

$\sin^2 x = \frac{1 - \cos(2x)}{2}$

Paso 2: Sustituir en la integral

$\int \sin^2 x \, dx = \int \frac{1 - \cos(2x)}{2} \, dx$

Paso 3: Separar términos

$= \frac{1}{2}\int 1 \, dx - \frac{1}{2}\int \cos(2x) \, dx$

Paso 4: Integrar

$= \frac{1}{2}x - \frac{1}{2} \cdot \frac{\sin(2x)}{2} + C$

Paso 5: Simplificar

$= \frac{x}{2} - \frac{\sin(2x)}{4} + C$

Respuesta: $\int \sin^2 x \, dx = \frac{x}{2} - \frac{\sin(2x)}{4} + C$

Verificación por Derivación

¡Siempre verifica tu respuesta!

Para verificar que $\int f(x) \, dx = F(x) + C$ es correcta:

$$\frac{d}{dx}[F(x) + C] = f(x)$$

Si obtienes $f(x)$ al derivar $F(x) + C$, entonces tu integral es correcta.

Ejemplo 11: Verificación Completa

Problema: Verifica que $\int (3x^2 - 2x + 1) \, dx = x^3 - x^2 + x + C$

Paso 1: Derivar la respuesta

$\frac{d}{dx}[x^3 - x^2 + x + C] = 3x^2 - 2x + 1$

Paso 2: Comparar con el integrando

El integrando original era $3x^2 - 2x + 1$ ✓

Paso 3: Conclusión

Como la derivada coincide con el integrando, la integral es correcta.

Verificación: ✓ Correcta

Ejercicio 1

Encuentra una antiderivada de $f(x) = 6x^2$

Solución: $F(x) = 2x^3 + C$

Verificación: $\frac{d}{dx}[2x^3 + C] = 6x^2$ ✓

Ejercicio 2

Encuentra todas las antiderivadas de $f(x) = 4x - 3$

Solución: $F(x) = 2x^2 - 3x + C$

Verificación: $\frac{d}{dx}[2x^2 - 3x + C] = 4x - 3$ ✓

Ejercicio 3

Verifica que $F(x) = x^4 - 2x^2 + 5$ es una antiderivada de $f(x) = 4x^3 - 4x$

Solución: $\frac{d}{dx}[x^4 - 2x^2 + 5] = 4x^3 - 4x$ ✓

Por lo tanto, $F(x)$ es una antiderivada de $f(x)$.

Ejercicios 1.2: Integrales Inmediatas

Ejercicio 4

Evalúa $\int 5x^3 \, dx$

Solución: $\int 5x^3 \, dx = 5 \cdot \frac{x^4}{4} + C = \frac{5x^4}{4} + C$

Ejercicio 5

Evalúa $\int (x^2 + 3x - 2) \, dx$

Solución: $\int (x^2 + 3x - 2) \, dx = \frac{x^3}{3} + \frac{3x^2}{2} - 2x + C$

Ejercicio 6

Evalúa $\int (\sin x + \cos x) \, dx$

Solución: $\int (\sin x + \cos x) \, dx = -\cos x + \sin x + C$

Ejercicio 7

Evalúa $\int \frac{1}{x^2} \, dx$

Solución: $\int x^{-2} \, dx = \frac{x^{-1}}{-1} + C = -\frac{1}{x} + C$

Ejercicios 1.3: Técnicas Básicas

Ejercicio 8

Evalúa $\int (x + 1)^3 \, dx$

Solución: $(x + 1)^3 = x^3 + 3x^2 + 3x + 1$

$\int (x^3 + 3x^2 + 3x + 1) \, dx = \frac{x^4}{4} + x^3 + \frac{3x^2}{2} + x + C$

Ejercicio 9

Evalúa $\int \frac{x^3 - 2x + 1}{x} \, dx$

Solución: $\frac{x^3 - 2x + 1}{x} = x^2 - 2 + \frac{1}{x}$

$\int (x^2 - 2 + \frac{1}{x}) \, dx = \frac{x^3}{3} - 2x + \ln|x| + C$

Ejercicio 10

Evalúa $\int \cos^2 x \, dx$

Solución: $\cos^2 x = \frac{1 + \cos(2x)}{2}$

$\int \frac{1 + \cos(2x)}{2} \, dx = \frac{x}{2} + \frac{\sin(2x)}{4} + C$
