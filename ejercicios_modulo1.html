1.1 Concepto de Antiderivada

Ejercicio 1

Básico

Problema: Encuentra una antiderivada de $f(x) = 6x^2$

Paso 1:

Identificar que necesitamos una función cuya derivada sea $6x^2$

Paso 2:

Recordar que $\frac{d}{dx}[x^3] = 3x^2$, entonces $\frac{d}{dx}[2x^3] = 6x^2$

Paso 3:

Por lo tanto, una antiderivada es $F(x) = 2x^3$

Respuesta: $F(x) = 2x^3 + C$

Ejercicio 2

Básico

Problema: Encuentra todas las antiderivadas de $f(x) = 4x - 3$

Paso 1:

Buscar una función cuya derivada sea $4x - 3$

Paso 2:

$\frac{d}{dx}[2x^2] = 4x$ y $\frac{d}{dx}[3x] = 3$

Paso 3:

Entonces $\frac{d}{dx}[2x^2 - 3x] = 4x - 3$

Respuesta: $F(x) = 2x^2 - 3x + C$

Ejercicio 3

Intermedio

Problema: Verifica que $F(x) = x^4 - 2x^2 + 5$ es una antiderivada de $f(x) = 4x^3 - 4x$

Paso 1:

Derivar $F(x) = x^4 - 2x^2 + 5$

Paso 2:

$F'(x) = 4x^3 - 4x + 0 = 4x^3 - 4x$

Paso 3:

Comparar con $f(x) = 4x^3 - 4x$ ✓

Verificación: ✓ Correcta

Ejercicio 4

Básico

Problema: ¿Cuál es la familia de antiderivadas de $f(x) = 8x$?

Paso 1:

Buscar una función cuya derivada sea $8x$

Paso 2:

$\frac{d}{dx}[4x^2] = 8x$

Paso 3:

Agregar la constante de integración

Respuesta: $F(x) = 4x^2 + C$

Ejercicio 5

Intermedio

Problema: Si $F(x) = 3x^3 - 2x + 1$ es una antiderivada de $f(x)$, encuentra $f(x)$

Paso 1:

Si $F(x)$ es una antiderivada de $f(x)$, entonces $f(x) = F'(x)$

Paso 2:

Derivar $F(x) = 3x^3 - 2x + 1$

Paso 3:

$F'(x) = 9x^2 - 2 + 0 = 9x^2 - 2$

Respuesta: $f(x) = 9x^2 - 2$

1.2 Integrales Inmediatas

Ejercicio 6

Básico

Problema: Evalúa $\int 5x^3 \, dx$

Paso 1:

Aplicar la regla de la potencia: $\int x^n \, dx = \frac{x^{n+1}}{n+1} + C$

Paso 2:

$\int 5x^3 \, dx = 5 \cdot \frac{x^4}{4} + C$

Paso 3:

Simplificar: $\frac{5x^4}{4} + C$

Respuesta: $\int 5x^3 \, dx = \frac{5x^4}{4} + C$

Ejercicio 7

Básico

Problema: Evalúa $\int (x^2 + 3x - 2) \, dx$

Paso 1:

Separar términos: $\int x^2 \, dx + 3\int x \, dx - 2\int 1 \, dx$

Paso 2:

Aplicar regla de la potencia a cada término

Paso 3:

$= \frac{x^3}{3} + 3 \cdot \frac{x^2}{2} - 2x + C$

Respuesta: $\int (x^2 + 3x - 2) \, dx = \frac{x^3}{3} + \frac{3x^2}{2} - 2x + C$

Ejercicio 8

Básico

Problema: Evalúa $\int (\sin x + \cos x) \, dx$

Paso 1:

Separar términos: $\int \sin x \, dx + \int \cos x \, dx$

Paso 2:

Aplicar integrales trigonométricas básicas

Paso 3:

$= -\cos x + \sin x + C$

Respuesta: $\int (\sin x + \cos x) \, dx = -\cos x + \sin x + C$

Ejercicio 9

Intermedio

Problema: Evalúa $\int \frac{1}{x^2} \, dx$

Paso 1:

Reescribir como potencia: $\frac{1}{x^2} = x^{-2}$

Paso 2:

Aplicar regla de la potencia: $\int x^{-2} \, dx = \frac{x^{-1}}{-1} + C$

Paso 3:

Simplificar: $-\frac{1}{x} + C$

Respuesta: $\int \frac{1}{x^2} \, dx = -\frac{1}{x} + C$

Ejercicio 10

Intermedio

Problema: Evalúa $\int (2x^4 - 3x^2 + 1) \, dx$

Paso 1:

Separar términos: $2\int x^4 \, dx - 3\int x^2 \, dx + \int 1 \, dx$

Paso 2:

Aplicar regla de la potencia: $2 \cdot \frac{x^5}{5} - 3 \cdot \frac{x^3}{3} + x + C$

Paso 3:

Simplificar: $\frac{2x^5}{5} - x^3 + x + C$

Respuesta: $\int (2x^4 - 3x^2 + 1) \, dx = \frac{2x^5}{5} - x^3 + x + C$

Ejercicio 11

Básico

Problema: Evalúa $\int \sqrt{x} \, dx$

Paso 1:

Reescribir como potencia: $\sqrt{x} = x^{1/2}$

Paso 2:

Aplicar regla de la potencia: $\int x^{1/2} \, dx = \frac{x^{3/2}}{3/2} + C$

Paso 3:

Simplificar: $\frac{2x^{3/2}}{3} + C = \frac{2x\sqrt{x}}{3} + C$

Respuesta: $\int \sqrt{x} \, dx = \frac{2x\sqrt{x}}{3} + C$

Ejercicio 12

Intermedio

Problema: Evalúa $\int (3\sin x - 2\cos x + \sec^2 x) \, dx$

Paso 1:

Separar términos: $3\int \sin x \, dx - 2\int \cos x \, dx + \int \sec^2 x \, dx$

Paso 2:

Aplicar integrales trigonométricas básicas

Paso 3:

$= 3(-\cos x) - 2(\sin x) + \tan x + C$

Paso 4:

Simplificar: $-3\cos x - 2\sin x + \tan x + C$

Respuesta: $\int (3\sin x - 2\cos x + \sec^2 x) \, dx = -3\cos x - 2\sin x + \tan x + C$

1.3 Técnicas Básicas de Integración

Ejercicio 13

Intermedio

Problema: Evalúa $\int (x + 1)^3 \, dx$

Paso 1:

Expandir el binomio: $(x + 1)^3 = x^3 + 3x^2 + 3x + 1$

Paso 2:

Integrar término por término: $\int (x^3 + 3x^2 + 3x + 1) \, dx$

Paso 3:

Aplicar regla de la potencia: $\frac{x^4}{4} + 3 \cdot \frac{x^3}{3} + 3 \cdot \frac{x^2}{2} + x + C$

Paso 4:

Simplificar: $\frac{x^4}{4} + x^3 + \frac{3x^2}{2} + x + C$

Respuesta: $\int (x + 1)^3 \, dx = \frac{x^4}{4} + x^3 + \frac{3x^2}{2} + x + C$

Ejercicio 14

Intermedio

Problema: Evalúa $\int \frac{x^3 - 2x + 1}{x} \, dx$

Paso 1:

Dividir término por término: $\frac{x^3 - 2x + 1}{x} = x^2 - 2 + \frac{1}{x}$

Paso 2:

Integrar término por término: $\int x^2 \, dx - 2\int 1 \, dx + \int \frac{1}{x} \, dx$

Paso 3:

Aplicar reglas: $\frac{x^3}{3} - 2x + \ln|x| + C$

Respuesta: $\int \frac{x^3 - 2x + 1}{x} \, dx = \frac{x^3}{3} - 2x + \ln|x| + C$

Ejercicio 15

Avanzado

Problema: Evalúa $\int \sin^2 x \, dx$

Paso 1:

Usar identidad trigonométrica: $\sin^2 x = \frac{1 - \cos(2x)}{2}$

Paso 2:

Sustituir: $\int \frac{1 - \cos(2x)}{2} \, dx$

Paso 3:

Separar términos: $\frac{1}{2}\int 1 \, dx - \frac{1}{2}\int \cos(2x) \, dx$

Paso 4:

Integrar: $\frac{1}{2}x - \frac{1}{2} \cdot \frac{\sin(2x)}{2} + C$

Paso 5:

Simplificar: $\frac{x}{2} - \frac{\sin(2x)}{4} + C$

Respuesta: $\int \sin^2 x \, dx = \frac{x}{2} - \frac{\sin(2x)}{4} + C$

Ejercicio 16

Avanzado

Problema: Evalúa $\int \cos^2 x \, dx$

Paso 1:

Usar identidad trigonométrica: $\cos^2 x = \frac{1 + \cos(2x)}{2}$

Paso 2:

Sustituir: $\int \frac{1 + \cos(2x)}{2} \, dx$

Paso 3:

Separar términos: $\frac{1}{2}\int 1 \, dx + \frac{1}{2}\int \cos(2x) \, dx$

Paso 4:

Integrar: $\frac{1}{2}x + \frac{1}{2} \cdot \frac{\sin(2x)}{2} + C$

Paso 5:

Simplificar: $\frac{x}{2} + \frac{\sin(2x)}{4} + C$

Respuesta: $\int \cos^2 x \, dx = \frac{x}{2} + \frac{\sin(2x)}{4} + C$

Ejercicio 17

Intermedio

Problema: Evalúa $\int \frac{x^2 - 4}{x} \, dx$

Paso 1:

Dividir término por término: $\frac{x^2 - 4}{x} = x - \frac{4}{x}$

Paso 2:

Integrar término por término: $\int x \, dx - 4\int \frac{1}{x} \, dx$

Paso 3:

Aplicar reglas: $\frac{x^2}{2} - 4\ln|x| + C$

Respuesta: $\int \frac{x^2 - 4}{x} \, dx = \frac{x^2}{2} - 4\ln|x| + C$

Ejercicio 18

Intermedio

Problema: Evalúa $\int (2x - 1)^2 \, dx$

Paso 1:

Expandir el binomio: $(2x - 1)^2 = 4x^2 - 4x + 1$

Paso 2:

Integrar término por término: $\int (4x^2 - 4x + 1) \, dx$

Paso 3:

Aplicar regla de la potencia: $4 \cdot \frac{x^3}{3} - 4 \cdot \frac{x^2}{2} + x + C$

Paso 4:

Simplificar: $\frac{4x^3}{3} - 2x^2 + x + C$

Respuesta: $\int (2x - 1)^2 \, dx = \frac{4x^3}{3} - 2x^2 + x + C$
